# Angular Website

Angular 21 implementation for bachelor thesis framework comparison.

## Tech Stack

- Angular 21
- Angular Router
- TypeScript 5.9

## Scripts

```bash
npm install     # Install dependencies
npm start       # Development server
npm run build   # Production build
npm run deploy  # Deploy to GitHub Pages
```

## Structure

```
src/app/
├── api/         # API services
├── components/  # Reusable components
├── pages/       # Page components
└── app.routes.ts
```
